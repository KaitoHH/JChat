# JChat
a Java Chat Messager 

### Features
- Both Client and Server so that you can create a server on your local pc
- Basic funcion such as online/offline message, channel join/switching, whisper, etc.
- Windows jittering 
- enjoy:)

### Complie Environment
jDK 1.8.0_45
